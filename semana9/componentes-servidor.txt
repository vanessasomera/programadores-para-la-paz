Sistema operativo:
Es el software principal que administra los recursos del computador y permite ejecutar programas y servicios.

Node.js:
Es un entorno de ejecución que permite utilizar JavaScript fuera del navegador para crear aplicaciones servidor.

Aplicación server.js:
Contiene el código del servidor. Define las rutas y las respuestas que recibirán los usuarios cuando hagan solicitudes.

Puerto:
Es un canal de comunicación utilizado por las aplicaciones para recibir conexiones. En esta práctica se utiliza el puerto 3000.

Logs:
Son registros de eventos y actividades del sistema. Ayudan a monitorear el funcionamiento del servidor y detectar errores.
